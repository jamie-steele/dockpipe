# `packages/pipeon/apps/` — Pipeon host applications

First-party graphical host apps for Pipeon live here. They drive DockPipe and related package tools from the host, but they are not part of the DockPipe engine in `src/lib/` or `src/cmd/`.

| Directory | Tool |
|-----------|------|
| **`pipeon-desktop/`** | **Pipeon Desktop** — Tauri desktop shell for the Pipeon/code-server surface |
| **`pipeon-launcher/`** | **Pipeon Launcher** — Qt tray app for launching DockPipe-backed sessions |
