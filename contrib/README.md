# `contrib/` — editor & ecosystem add-ons

Optional integrations **outside** the DockPipe Go core. Each subdirectory is its own package (npm, etc.).

| Directory | What it is |
|-----------|------------|
| **[pipeon-vscode-extension/](pipeon-vscode-extension/)** | **Pipeon IDE** — VS Code extension (workspace / commands). Not the same binary as **Pipeon Launcher** in **[apps/pipeon-launcher/](../apps/pipeon-launcher/)**. |

Pipeon docs (fork playbook, shortcuts, harness): **[pipeon/](../pipeon/README.md)**.

See **[docs/core-tools.md](../docs/core-tools.md)** for how this fits next to DockPipe and DorkPipe.
