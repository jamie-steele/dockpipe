# Images (repository note)

Framework Dockerfiles that ship with dockpipe and are merged into user projects by **`dockpipe init`** live under **`templates/core/assets/images/`** in this repository.

This top-level **`images/`** directory is reserved for **repository-only** image experiments or docs if needed; the supported, bundled layout is **`templates/core/assets/images/<name>/Dockerfile`**.

See **[docs/templates-core-assets.md](../docs/templates-core-assets.md)** for classification and compliance notes.
