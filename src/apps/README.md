# `apps/` — DockPipe host UIs

First-party applications that **drive** DockPipe from the host. They do **not** replace the `dockpipe` CLI; they **invoke** it.

| Directory | Tool |
|-----------|------|
| **`pipeon-launcher/`** | **Pipeon Launcher** — DockPipe GUI: Qt 6 tray app, contexts, `dockpipe` subprocess |

**Pipeon** harness docs and the **VS Code extension** ship in first-party **`packages/pipeon/`** — **`packages/pipeon/resolvers/pipeon/README.md`**.
