# Framework images (`templates/core/assets/images/`)

Dockerfiles used by **`TemplateBuild`** and **`--isolate`** for template names (`base-dev`, `dev`, `claude`, `codex`, `vscode`, …). Build context is always the **repository root** (for **`COPY lib/entrypoint.sh`**).
